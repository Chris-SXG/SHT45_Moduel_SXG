# AirLibre_CO2  
## 專案介紹:
SHT45_Module_SXG是一個開源的小型CO2模組  
### 應用場所:  
一般室內環境，ex.居家、辦公室、健身房、餐廳、大賣場....
### 主要功能:  
* 可㩦帶(充電式 短時間使用)
* 長時間偵測(電源供應器 長時間使用)
* 無網路環境，裝置本身可保存最近1週內的數據
* 數據透過Wifi上傳雲端(保留) 
* 透過Android APP 連裝置 查看數據(保留)

### 技術文件:  
* 硬體方塊圖
* 主要元件
* 源始碼
* 線路圖
* PCB Layotu
* BOM
